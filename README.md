# FairFlow Backend

**Blockchain-Powered Financial Ledger System**

FairFlow Backend is the server-side component of a transparent and immutable financial transaction ledger built using blockchain technology.

**Developed as part of my final year project**

## Key Features
- Immutable transaction recording on Ethereum-compatible blockchain
- RESTful API for ledger operations
- Solidity smart contracts for core financial logic
- Node.js + Express backend
- Serverless deployment support (Vercel)

## Tech Stack
- Backend: Node.js, Express
- Blockchain: Solidity, ethers.js
- Deployment: Vercel
